Hybrid Quantum–Classical Machine Learning for Anomaly Detection

This project implements a hybrid quantum–classical machine learning framework
for anomaly detection using the NSL-KDD dataset.

Models implemented:
- Classical Random Forest
- Quantum Variational Classifier (VQC)
- Hybrid Quantum–Classical model
- Explainable AI (SHAP and sensitivity analysis)

The hybrid model achieved the best performance and improved attack detection
compared to individual models.
